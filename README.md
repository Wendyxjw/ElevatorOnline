# iView-project

This project is build for Vue.js 2 + vue-router + webpack2 + iView 2, just install and run.

## Install
```bush
// install dependencies
npm install
```
## Run
### Development
```bush
// For the first time, run init to create index.html
npm run dev
```
### Production(Build)
```bush
npm run build
```
### 本地打开build页面
```
npm install serve -g
cd dist
serve
```
